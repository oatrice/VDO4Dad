# Video Content Guidelines

This directory contains video files that are intended for offline viewing. The content is curated to ensure a positive and entertaining experience, focusing on humor and educational material. Below are the guidelines for managing the video content in this project.

## Content Categories

1. **Comedy Videos**
   - Examples: Mr. Bean, โน๊ตอุดม, สะเก็ดข่าว, คดีเด็ด, ก็มาดิครับ, ระเบิดเถิดเทิง, ชิงร้อยชิงล้าน
   - These videos should be light-hearted and suitable for all audiences.

2. **Relaxing Music**
   - Include instrumental music that promotes relaxation and a positive atmosphere.

3. **Educational and Informative Content**
   - Examples: ธรรมะ, พระอาจารย์สมชาย ฐานะวุฒโฒ, นานาสาระ, สารคดี, กฎแห่งกรรม, case study
   - This content should provide valuable insights and knowledge.

## Adding New Videos

To add new videos to the project:

1. Place the video file in the appropriate directory within `src/videos/`.
2. Update the `src/data/videos.json` file to include the new video object with the following properties:
   - `title`: The title of the video.
   - `description`: A brief description of the video content.
   - `filePath`: The relative path to the video file.

## Managing Existing Videos

- Review the content regularly to ensure it aligns with the project's guidelines.
- Remove any videos that do not meet the criteria or that may contain inappropriate content.

## Conclusion

This project aims to provide a curated selection of videos that entertain and educate. By following these guidelines, we can ensure a positive viewing experience for everyone.
